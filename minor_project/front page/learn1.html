Q1

What is DSA?

Data structure is a way of defining, storing & retriving of data in a structural & systemetic way. A data structure may contain different type of data items.

Q2

What is a linked-list??

A linked-list is a list of data-items connected with links i.e. pointers or references.

Q3

What is stack?

In data-structure, stack is an Abstract Data Type (ADT) used to store and retrieve values in Last In First Out method.

Q4

Why do we use stacks?

Stacks are used where we need to access data in the reverse order or their arrival. Stacks are used commonly in recursive function calls, expression parsing, depth first traversal of graphs etc.

Q5

What is a queue?

Queue is opened at both end. One end is always used to insert data and the other is used to remove data. Queue follows FIFO, i.e., the data item stored first will be accessed first.

Q6

What is linear searching?

Linear search tries to find an item in a sequentially arranged data type. These sequentially arranged data items are accessible in incrementing memory location.

Q7

What is binary search?

A binary search works only on sorted lists or arrays. First the middle is compared. This search first compares the target value to the mid of the list. If it isn't found, then it takes another decision.

Q8

What is bubble sort?

Bubble sort is comparison based algorithm in which each pair of adjacent elements is compared and elements are swapped if they are not in order.

Q9

What is shell sort?

Shell sort can be said a variant of insertion sort. Shell sort divides the list into smaller sublist based on some gap variable and then each sub-list is sorted using insertion sort.

Q10

How DFS works?

Depth First Search algorithm(DFS) traverses a graph in a depthward motion and uses a stack to remember to get the next vertex to start a search when a dead end occurs in any iteration.

Q11

How BFS works?

Breadth First Search (BFS) traverses a graph in a breadthwards motion and uses a queue to remember to get the next vertex to start a search when a dead end occurs in any iteration.

Q12

What is a binary tree?

A binary tree has a special condition that each node can have two children at maximum.

Q13

What is a BST?

A binary search tree is a type of binary treewhere a node's left child must have value less than its parent's value and node's right child must have value greater than it's parent value.

Q14

What is tree traversal?

Tree traversal is a process to visit all the nodes of a tree. Because, all nodes are connected via edges (links) we always start from the root (head) node.

Q15

What is an AVL Tree?

AVL trees are height balancing binary search tree. AVL tree checks the height of left and right sub-trees and assures that the difference is not more than 1. This difference is called Balance Factor.
